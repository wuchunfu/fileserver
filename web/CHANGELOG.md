# 2021-10-29

- 新增 [`i18n`](https://vue-i18n.intlify.dev/introduction.html) 支持

### 如何使用？

语言包存放在 `src/i18n/languages` 文件夹下，每种语言一个文件

> 为保证 `element-plus` 正确国际化，请确保 `src/i18n/languages` 文件夹下的语言包名与 `element-plus` 的语言包名一致
